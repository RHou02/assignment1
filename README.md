# Assignment 1

## Overview

This repository contains the work for Assignment 1, which includes two main parts:

- **Q1 – Frailty Analysis**
- **Q2 – Data Visualization**

Each question has its own folder containing code, reports, and related materials.

---

## Q1 – Frailty Analysis

The analysis for Question 1 is located in the `q1_frailty` folder.

- The main findings are documented in:

  `q1_frailty/reports/findings.md`

- All source code and supporting files used for the analysis are organized within the same directory.

---

## Q2 – Data Visualization

The visualization analysis for Question 2 is located in the `q2_visualization` folder.

- The written findings are documented in:

  `q2_visualization/reports/findings.md`

- All generated figures (V1–V5) are stored inside the `reports` folder within `q2_visualization`.

- The dataset and visualization source code are also contained within the `q2_visualization` directory.
